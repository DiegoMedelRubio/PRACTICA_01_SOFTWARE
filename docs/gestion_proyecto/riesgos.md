# Matriz de riesgos del proyecto

| Riesgo | Impacto | Probabilidad | Mitigación |
|------|------|------|------|
| Falta de definición clara de requisitos | Alto | Medio | Revisiones frecuentes entre el equipo |
| Falta de comunicación entre miembros | Medio | Medio | Uso constante de GitHub Issues y comentarios |
| Cambios en los requisitos del proyecto | Medio | Alto | Revisar y actualizar backlog |
| Falta de tiempo para completar tareas | Alto | Medio | Planificar tareas por sprints |
