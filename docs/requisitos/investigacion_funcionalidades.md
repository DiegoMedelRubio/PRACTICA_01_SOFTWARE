# Investigación de funcionalidades - App para pacientes con patologías cardiacas

Tras un análisis inicial, se identifican las siguientes funcionalidades clave:

## 1. Registro de datos de salud
- Registro de frecuencia cardíaca
- Registro de presión arterial
- Registro de síntomas

## 2. Recordatorios de medicación
- Configuración de horarios
- Notificaciones automáticas
- Confirmación de toma de medicación

## 3. Historial médico
- Visualización por fechas
- Filtros por tipo de dato
- Resumen gráfico

## 4. Sistema de alertas
- Detección de valores fuera de rango
- Aviso visual inmediato
- Recomendación de contacto médico

## 5. Acceso profesional sanitario
- Visualización de datos del paciente
- Seguimiento histórico
- Acceso seguro mediante autenticación
