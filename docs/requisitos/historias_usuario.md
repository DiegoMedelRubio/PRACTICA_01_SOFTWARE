# Historias de Usuario

## HU-01
Como paciente  
Quiero registrar mi frecuencia cardíaca diaria  
Para poder llevar un seguimiento de mi estado de salud.

Criterios de aceptación:
- El usuario puede introducir manualmente su frecuencia cardíaca.
- Los datos quedan almacenados en su perfil.
- Se puede consultar el historial por fechas.

---

## HU-02
Como paciente  
Quiero registrar mis síntomas diarios  
Para informar mejor a mi médico sobre mi estado.

Criterios de aceptación:
- El usuario puede escribir síntomas.
- Los síntomas quedan asociados a una fecha.
- Se pueden visualizar en el historial.

---

## HU-03
Como paciente  
Quiero recibir recordatorios de medicación  
Para no olvidar tomar mis medicamentos.

Criterios de aceptación:
- Se pueden configurar horarios de medicación.
- Se generan notificaciones automáticas.
- El usuario puede marcar la medicación como tomada.

---

## HU-04
Como paciente  
Quiero consultar mi historial de datos médicos  
Para revisar mi evolución.

Criterios de aceptación:
- Se muestran registros ordenados por fecha.
- Se pueden filtrar por tipo de dato.
- La información se muestra de forma clara.

---

## HU-05
Como paciente  
Quiero recibir alertas cuando mis valores sean anormales  
Para actuar rápidamente ante posibles riesgos.

Criterios de aceptación:
- El sistema detecta valores fuera de rango.
- Se muestra una alerta en pantalla.
- Se recomienda contactar con un profesional.

---

## HU-06
Como profesional sanitario  
Quiero acceder a los datos del paciente  
Para poder realizar un seguimiento médico adecuado.

Criterios de aceptación:
- El médico puede visualizar los registros del paciente.
- El acceso está protegido mediante autenticación.
- La información se muestra de forma clara y organizada.

---

## HU-07
Como paciente  
Quiero poder iniciar sesión de forma segura  
Para proteger mis datos personales.

Criterios de aceptación:
- El sistema requiere usuario y contraseña.
- Los datos están protegidos.
- El usuario puede cerrar sesión.

---

## HU-08
Como paciente  
Quiero exportar un informe en PDF  
Para compartirlo con mi médico.

Criterios de aceptación:
- Se puede generar un archivo PDF.
- El informe incluye datos relevantes.
- El archivo se puede descargar.
