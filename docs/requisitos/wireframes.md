# Wireframes - Diseño de Pantallas

## 1. Pantalla de Login

Elementos:
- Campo de usuario
- Campo de contraseña
- Botón "Iniciar sesión"
- Enlace "¿Olvidaste tu contraseña?"

Descripción:
Pantalla inicial que permite al usuario autenticarse de forma segura.

---

## 2. Pantalla Principal (Dashboard)

Elementos:
- Resumen de frecuencia cardíaca
- Últimos síntomas registrados
- Próxima medicación
- Botón "Registrar datos"
- Botón "Ver historial"

Descripción:
Pantalla principal donde el paciente visualiza un resumen rápido de su estado.

---

## 3. Pantalla de Registro de Datos

Elementos:
- Campo para frecuencia cardíaca
- Campo para síntomas
- Botón "Guardar"

Descripción:
Permite al paciente introducir nuevos datos de salud.

---

## 4. Pantalla de Historial

Elementos:
- Lista de registros por fecha
- Filtros por tipo de dato
- Opción de exportar en PDF

Descripción:
Permite consultar los datos registrados anteriormente.

---

## 5. Pantalla de Alertas

Elementos:
- Mensaje de alerta
- Indicador visual (color rojo)
- Recomendación de acción

Descripción:
Se muestra cuando el sistema detecta valores fuera de rango.

---

## 6. Pantalla Profesional Sanitario

Elementos:
- Lista de pacientes
- Acceso a datos individuales
- Historial detallado

Descripción:
Permite al profesional sanitario consultar la información del paciente.
